== Description ==

LatePoint is a powerful and user-friendly appointment scheduling plugin for WordPress that simplifies booking management for businesses of all sizes. It offers a variety of features, including an intuitive admin dashboard, responsive design for mobile devices, and seamless integration with popular services like Google Calendar and Zoom. The plugin supports multiple booking methods, such as using shortcodes, block elements, or triggers attached to existing elements, allowing you to customize how bookings are made on your site.

LatePoint also provides a range of add-ons for extended functionality, such as handling payments through various gateways, automated reminders via SMS or email, and allowing agents and customers to manage appointments through personalized dashboards. The plugin is highly flexible, making it suitable for developers to create custom add-ons and extend its capabilities

== For Developers ==
We use [Codekit](https://codekitapp.com/) to compile and minify assets. There is a config.codekit3 file which automatically applies settings to a codekit compiler. Assets are available on our [github repository](https://github.com/latepoint/latepoint) in latepoint/lib/assets/ folder.
